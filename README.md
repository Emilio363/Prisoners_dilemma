# Prisoners_dilemma
Questo progetto nasce da un progetto richiesto per un mio corso universitario. L'intento è quello di ricreare i risultati ottenuti
nel paper [historical strategy information](https://www.nature.com/articles/s41598-022-26890-9) che descrive come la memoria
di giochi passati, possa influenzare le strategie future nel dilemma del prigioniero.

Attualmente l'unico progetto completamente funzionante è quello scritto in C, gli altri necessitano ancora di revisione.

I dati ottenuti dal progetto in C non sono comunque coerenti con quanto descritto dal paper. Si spera in una presta revisione del
codice per ottenere dati corretti.
